## Correlated subquery

[Source](https://stackoverflow.com/a/77586936/22203726)
```surql
SELECT 
    id, 
    name, 
    favouriteDay, 
    (select motto from motto where day = $parent.favouriteDay)[0].motto as motto 
FROM user;
```
